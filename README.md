# Data-Pipeline-with-Python-SQL-Airflow
## Objective
Use Apache Airflow to automate an end-to-end data pipeline that analyzes daily sales revenue from a PostgreSQL database.



## What You Will Do
You will build a pipeline that:

- Extracts sales data from a PostgreSQL database using SQL.

- Calculates total revenue per day.

- Visualizes the daily revenue as a time series plot.

- Automates the workflow using Apache Airflow.

## Tools You May Use
- Python

- SQL

- PostgreSQL

- Apache Airflow

- Data visualization library (Matplotlib)
